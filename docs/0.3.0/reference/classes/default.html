animate

  • animate(obj: number | Chart | AnimTarget, opt?: Duration | Options): Promise<default>
    • Defined in vizzu.d.ts:647

    Initiates an animation to the new chart states passed as the first argument. If there is a currently running animation, all subsequent calls will schedule the corresponding animation after the end of the previous one.

    The new chart state can be a full state specifier object with data, config and style, or a single chart config object. It accepts also a chart snapshot acquired from a previous state using the store() method.

    The optional second parameter specifies the animation options. Second option can be a scalar value too, setting the overall animation duration.

    The animation will be initiated in the next cycle of the JS event loop. The method returns a promise, which will resolve when the animation is finished.

    Parameters

    • obj: number | Chart | AnimTarget
    • Optional opt: Duration | Options

    Returns Promise<default>
