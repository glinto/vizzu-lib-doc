Interface Chart

The config contains all the parameters needed to render a particular static chart or a state of an animated chart.

Optional channels

channels: object
  • Defined in vizzu.d.ts:144

List of the chart's channels. A data series name or a list of data series names can be used as a short-hand alternatively to the channel configuration object to set data series for the channel. Setting a channel to null will remove all data series from the channel.

Type declaration

  • Optional color?: Channel | SeriesList

    Parameters for markers' base color. The marker's effective color is also affected by the lightness channel.

  • Optional label?: Channel | SeriesList

    Parameters for the content of the markers' labels.

  • Optional lightness?: Channel | SeriesList

    Parameters for markers' lightness.

  • Optional noop?: Channel | SeriesList

    Splits the markers as all the other channels, but will not have an effect on the markers appearence.

  • Optional size?: Channel | SeriesList

    Parameters for markers' size, effective only for Circle and Line geometry affecting the circle area or the line width respectively.

  • Optional x?: Channel | SeriesList

    Parameters for X-axis determine the position of the markers on the x (or angle for the polar coordinate system) axis. Note: leaving x and y channels empty will result in a "without coordinates" chart.

  • Optional y?: Channel | SeriesList

    Parameters for Y-axis, determine the position of the markers on the y (or radius for the polar coordinate system) axis.

Optional legend

legend: "color" | "lightness" | "size"
  • Defined in vizzu.d.ts:175

Specifies which channel should be detailed on the legend. If set to null, the legend will not be shown and will not take up any space in the chart layout.

Optional orientation

orientation: "horizontal" | "vertical"
  • Defined in vizzu.d.ts:186

If both axes represent continuous data, this parameter sets the orientation of the chart, meaning to which axis the graphical elements are oriented to.

Optional reverse

reverse: boolean
  • Defined in vizzu.d.ts:193

Optional rotation

rotation: number
  • Defined in vizzu.d.ts:180

Rotates the plot area by the specified angle in degree.

Optional sort

sort: "none" | "byValue"
  • Defined in vizzu.d.ts:191

'none': markers are sorted in the order as the corresponding data appear in the data set. 'byValue': markers will be sorted by the corresponding continuous data (if present) in decreasing order.

Optional split

split: boolean
  • Defined in vizzu.d.ts:198

If set, markers will be aligned by the categories instead of getting stacked.

Optional title

title: string
  • Defined in vizzu.d.ts:171

This title is shown at the top of the chart. If set to null, the Title will not be shown and will not take up any space in the chart layout.

Optional tooltip

tooltip: number
  • Defined in vizzu.d.ts:201

The id of the tooltiped marker or null to hide the active tooltip.
